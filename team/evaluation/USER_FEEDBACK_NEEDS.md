# SBEats User Feedback Needs

## 1. Restaurant Discovery Feature

### Main Feedback Need  
Evaluate how effectively users can find restaurants that match their preferences.

### What We Want Feedback On  
- Ease of finding a restaurant  
- Usefulness of search bar
- Overall satisfaction with the discovery experience  

## 2. Save Restaurants Feature

### Main Feedback Need  
Determine whether the save feature is valuable and encourages future engagement.

### What We Want Feedback On  
- UI experience of using the save button
- Ease and usefulness of picking the different categories
- Saving and unsaving a certain restaurant

## 3. Overall Product Experience

### Main Feedback Need  
Assess general usability, perceived value, and likelihood of continued use.

### What We Want Feedback On  
- Ease of navigation throughout the app  
- Visual design and UI interface
- Biggest frustrations with the UI experience
- Additional features we should add or remove

## 4. Home Screen A/B Test: Map vs Explore Page

### Main Feedback Need  
Identify which home screen design better supports engagement and restaurant discovery.

### What We Want Feedback On  

**Option A: Map as Home Screen**
- Ease of finding nearby restaurants  
- Speed of decision-making  
- Preference for visual location-based browsing  

**Option B: Explore Page as Home Screen**
- Ease of browsing restaurant lists  
- Preference for structured discovery
